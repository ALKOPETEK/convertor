Этота совокупность программ позволяет заменить все фотографии в фиде любого вормата в формат .jpg.
Реализовано через cloudinary.com c помощью API. Т.е. вам не нужно загружать фотографии на сервер вручную. Все автоматизированно и бесплатно, но есть огранечения по количеству загружаемых фотографий в cloudinary.com.

Я разделил разделил это действие на 4 разные программы, для более простого понимания принципа работы и контроля на каждом этапе.

Переместите папку из загрузок на диск "C"
Вам нужно ваш фид назвать feed.xml и положить в одну папку с программами.

Далее по шагам:

1. Запускаем программу download_images.py - она скачивает все фотографии из фида и сохраняет в папку downloaded_webp (если у вас не установлен питон - установите)
Для проверки можно открыть папку downloaded_webp и посмотреть количество фотографий

2. Следующий шаг: конвертация всех изображений в JPG, включая .webp, .png и любые другие подходящие форматы.
        1. Открываем convert_to_jpg.py открываем терминал и устанавливаем pip install pillow
        2. Запускаем программу convert_to_jpg.py
Сконвертированные фотографии сохранятся в файле converted_jpg. Так же для проверки можно открыть папку converted_jpg и посмотреть количество фотографий

3. Шаг 3: Загрузка JPG-фотографий в Cloudinary, с сохранением связи между исходным URL и новым.
        1. Перед запуском, создайте аккаунт на https://cloudinary.com (это бесплатно) и получите:
                cloud_name
                api_key
                api_secret
            Они будут использоваться для авторизации.

        2. Установка зависимости: открываем терминал и устанавливаем pip install cloudinary
        3. Запускаем программу cloudinary_download.py
    Что делает скрипт:
    Загружает .jpg в папку tilda_uploads в твоем Cloudinary.

    Сохраняет JSON-файл url_mapping.json

**Если выбивает ошибку проверь, что установил cloudinary в ту самую версию 3.12, которую ты используешь**
**C:/Users/........./Python/Python312/python.exe -m pip install cloudinary**

4. Замена ссылок в фиде на новые из Cloudinary. Просто запускаем replace_links_in_feed.py
